Simplemd

A simple Lennard-Jones molecular dynamics software

Download this project as a .zip file Download this project as a tar.gz file

DESCRIPTION

This is simplemd, a very simple molecular dynamics software which can be used to simulate Lennard-Jones systems.

It has been written with the purpose of teaching molecular dynamics and of being easily readable.
